# Daily Self-Reflection

## 核心频率
**重要教训随时记 + 每天复盘**

- 随时记：任何失败/教训 → 立即写 memory/YYYY-MM-DD.md
- 每天复盘：heartbeat 时回答自省问题

## 自省流程

### 1. 今日三问
每天必须回答：
1. **时间浪费**：今天有没有做无效功？哪个任务？为什么？
2. **关键异议**：今天在哪个决策点提供了人类没考虑到的重要意见？
3. **偏好学习**：从人类这次决策/反馈中学到了什么新逻辑？

### 2. 即时记录
- 重要教训 → 立即写 `memory/YYYY-MM-DD.md`
- 失败反思 → 记清问题和解决方案
- 人类偏好 → 更新 USER.md 或 CLARITY.md

### 3. 每周复盘
- 扫描 `memory/YYYY-MM-DD.md`
- 提炼关键心得 → 写 MEMORY.md
- 识别重复错误 → 想解决方案
